# 基本操作



## 1、拉取镜像

~~~
docker pull tomcat:tag版本
~~~



## 2、启动镜像

~~~
docker run 8080:8080 tomcat
~~~



## 3、Dockerfile定制镜像

新建dockerfile文件

基本格式：

~~~shell
FROM 镜像名
WORKDIR "工作路径"
RUN shell命令
~~~

构建镜像命令：

~~~shell
docker build -t 镜像名 Dockerfile目录
~~~

其他命令：

~~~shell
COPY 上下文环境路径 宿主机路径
# ADD命令会自动下载url文件，解压缩tar包；但是行为未知，官方不推荐使用
ADD tar包/url 宿主机路径
~~~
